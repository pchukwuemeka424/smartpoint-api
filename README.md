# SmartPoint API

Backend API for SmartPoint POS System

## Features

- User authentication and authorization
- Customer management
- Inventory/Item management
- Sales processing
- Transaction tracking
- Financial reporting
- AI-powered insights
- Cashier management

## Tech Stack

- Node.js
- Express.js
- MongoDB/Database

## Getting Started

### Installation

```bash
npm install
```

### Running the Server

```bash
npm start
```

## API Routes

- `/api/auth` - Authentication endpoints
- `/api/customers` - Customer management
- `/api/items` - Inventory management
- `/api/sales` - Sales processing
- `/api/finance` - Financial reporting
- `/api/cashiers` - Cashier management
- `/api/ai` - AI features
- `/api/settings` - System settings
